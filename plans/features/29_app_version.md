# 29 Application version

## Brainstorming

The application version is declared in `package.json` as usual. Each packaged `EPUB` that includes `SOURCE.zip` has in its `settings.json` a version string corresponding to the version of the app that was used to package the EPUB file. If the version of the app that is running is differnet from the version listed in `settings.json` the user is notified. If the running version of the app is greater than the version used to package the EPUB the user has the option to update the embedded version of the `EDITME.html` app to the running version.
