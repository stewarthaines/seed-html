# 28 First Run User Experiance

## Brainstorming

This feature considers the first-run experience for app users who haven't encountered it before. It provides features for first-run when there are no workspaces in the file storage, and present different user options depending on whether the app is served from http (or https) or loaded locally with file: url. In case the app is online and has access to archive.org the user is presented with a set of sample EPUB files from a particular archive.org collection. If served from a file: url the assumption is that the EDITME.html app has been extracted from an existing EPUB file and the user will load (via file input select, or drag and drop) the EPUB file from which the app was extracted.
